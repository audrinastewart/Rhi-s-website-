# Rhi's Personal Website ✨

A custom-built personal website for my friend Rhi. This project is designed to be lightweight, fast, and responsive.

## 🚀 Live Site
This website is automatically deployed and hosted on **Vercel**.

## 🛠️ Built With
- **HTML5** (Structure and semantic markup)
- **CSS3** (Styling, layout, and responsive design)
- **Vanilla JavaScript** (Interactive features)

## 💻 Local Development

Since this project consists entirely of vanilla web technologies, no build step or node modules are required! To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/audrinastewart/Rhi-s-website-.git
   ```
2. Open `Index.html` in your preferred web browser.
   *Tip: For the best experience while making changes, use an extension like **Live Server** in VS Code to auto-reload the page on save.*

## 📁 Project Structure

- `Index.html` & `Home.html`: Main HTML pages
- `style.css`: All custom styling for the site
- `script.js`: Client-side logic and interactivity
- `README.md`: Project documentation
